## 🚀 快速开始

### 1. 环境准备

```bash
# 切换到预览版分支
git checkout feature/native-openai-support

# 更新依赖
pip install -r requirements.txt
```

### 2. 配置Google AI

```bash
# 在 .env 文件中添加
echo "GOOGLE_API_KEY=your_api_key_here" >> .env
```

### 3. 验证安装

```bash
# 测试Google AI功能
python tests/test_gemini_simple.py

# 启动Web界面
streamlit run web/app.py
```

### 4. 体验新功能

- 🎯 在Web界面中选择Google AI模型
- 🔧 配置自定义OpenAI端点
- 📊 运行股票分析测试新功能
